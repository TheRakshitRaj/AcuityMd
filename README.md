# AcuityMD Website Clone

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

A responsive, high-fidelity front-end clone of the AcuityMD website. This project was developed from scratch to practice building clean, modern B2B SaaS interfaces, focusing on professional typography, data-oriented layouts, and a responsive user experience.

## 🎯 Features

* **B2B SaaS Aesthetic:** Faithfully recreates the clean, trustworthy, and professional UI typical of enterprise healthcare technology platforms.
* **Mock Data Layouts:** Uses advanced CSS Grid and Flexbox to structure complex feature showcases, mock dashboard interfaces, and data-centric components.
* **Fully Responsive:** Ensures a seamless and legible experience across all screen sizes, from mobile phones to large desktop monitors.
* **Clean Code Architecture:** Emphasizes semantic HTML for accessibility and highly organized, modular CSS for easy maintenance.
* **Zero Dependencies:** Built entirely with vanilla HTML5 and CSS3—no external styling frameworks (like Tailwind) or JavaScript libraries were used.

## 🚀 Technologies Used

* **HTML5** (Semantic structure and accessibility)
* **CSS3** (Flexbox, Grid, Custom Properties, Media Queries)

## 📁 Getting Started

To view this project locally on your machine, follow these simple steps:

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Safari, Edge) to view this project. No Node.js, build tools, or terminal commands are required.

### Installation

1. Download the project files as a ZIP archive to your local machine and extract the folder.
2. Navigate into the extracted project directory.
3. Double-click the `index.html` file to open it directly in your default web browser. 
   * *(Note: If you are using a code editor like VS Code, opening the project with the **Live Server** extension is recommended for real-time updates as you explore the code).*
